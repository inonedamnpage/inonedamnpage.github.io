Laying out elements in CSS is possibly one of the tricky parts of the language. Here we discuss several ways to do it.

Flexbox Layout

This is one of the most simplest ways to lay out content in CSS. To display content in flexbox layout, you have to add display property flex in one of the parent containers. The browser will then stack all containers side by side (or top bottom) according to the width or height provided.

The Flex Wrap Property

It determines how the columns should stack up incase the width is more than the row width. If it’s wrap it goes to the next row. Likewise you can also fix the directions using other values.

Overflow

Incase the text or content inside a column overflows, you can add a property to add a scrollbar.
